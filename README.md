# Elemental
  El proyecto consiste en la presentacion de dos conjuntos de simbolos utilizados para representar a los 4 elementos en diferentes contextos (en este caso, con los simbolos alquimicos de los elementos, y los simbolos ficticios utilizados para representar a las naciones de los elementos en "Avatar: the last airbender"), estos simbolos estan ubicado en 4 puntos del cuerpo y su cambio con la respectiva contraparte se puede observar cuando la persona se mueve verticalmente en la pantalla.
